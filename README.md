# dotfiles

Daniel Mathiot personal dotfiles

## Installation Method 

### Tmux, Vim,, Zsh

1. Install Brew
2. Execute command:
```bash

brew install tmux vim zsh

```
3. Install Oh-my-Zsh
4. Copy the files in home directory 
5. Apply changes:
    * Source zshrc
